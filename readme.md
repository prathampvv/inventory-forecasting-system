# 🍷 Inventory Forecasting & Par Level Recommendation System

A Python-based inventory forecasting system that predicts weekly beverage demand and recommends optimal inventory (par) levels for a multi-bar hotel chain. The project helps reduce stockouts, minimize overstocking, and improve inventory planning using historical consumption data and time-series forecasting.

---

## 📌 Project Overview

Managing inventory across multiple hotel bars is challenging due to fluctuating customer demand. Overstocking increases storage costs and product waste, while understocking leads to stockouts and poor customer experience.

This project analyzes historical beverage consumption data, forecasts future demand using time-series forecasting, and recommends optimal inventory levels (Par Levels) for every bar-item combination.

---

## 🎯 Problem Statement

Hotel bars often face:

- Frequent stockouts of popular beverages
- Overstocking of slow-moving inventory
- Manual inventory planning
- Increased operational costs
- Poor purchasing decisions

The objective of this project is to automate inventory planning by forecasting demand and recommending data-driven inventory levels.

---

## 🚀 Solution

The system performs the following tasks:

- Cleans and preprocesses historical consumption data
- Aggregates daily sales into weekly demand
- Performs exploratory data analysis
- Forecasts future demand using Exponential Smoothing
- Calculates recommended Par Levels
- Visualizes inventory insights using charts
- Exports recommendations into Excel

---

## 📊 Dataset

The project uses two Excel datasets.

### 1. Consumption Dataset

Contains historical beverage consumption data including:

- Date Time Served
- Bar Name

- Consumed (ml)

### 2. Inventory Dataset

Stores recommended inventory par levels generated by the forecasting model.

---

## 📉 Forecasting Model

The project uses **Exponential Smoothing** for demand forecasting.

Why Exponential Smoothing?

- Suitable for time-series forecasting
- Captures demand trends effectively
- Simple and computationally efficient
- Works well for inventory planning

---

## 📦 Par Level Calculation

The recommended inventory level is calculated using:

Par Level = Average Weekly Demand + (1.5 × Standard Deviation)

This safety buffer helps reduce the risk of stockouts while avoiding unnecessary overstocking.
